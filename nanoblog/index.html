"Vista" ist auch keine Chance für Linux

In einem Editorial auf der Linux-News Seite Pro-Linux schreibt Benjamin Quest über die Chancen von Linux durch die Einführung von Windows Vista.

Ähnliche Gedanken hatte ich auch schon bei der Einführung von Windows XP und die Realität zeigt uns, dass es zu keinem Nennenswerten Anstieg der Verbreitung von Linux durch die Einführung von XP gab.

Ich habe auch nie verstanden, warum Leute ein riesiges Problem damit haben sich an eine neue Software zu gewöhnen, wenn sie auch gleichzeitig einen anderen Namen trägt. z.B. wenn man versucht jemandem die Verwendung von OpenOffice schmackhaft zu machen gibt es jede Menge Leute, die OpenOffice ständig mit ihrem bisherigen Office vergleichen und sich über jeden geänderten Menupunkt oder jedes anders aussehende Icon beschweren. Wenn Microsoft allerdings von einer Version zur nächsten das komplette Verhalten auf den Kopf stellt, wird das komischerweise von den meisten kommentarlos hingenommen. Und genauso wird es auch mit Windows Vista® laufen.
